# natus-txt-parser

Parser general para archivos `.txt` exportados por Natus. El script detecta bloques, pares `clave=valor`, señales largas divididas en varias líneas y exporta los datos a formatos utilizables en Python y MATLAB.

## Requisitos

Este proyecto necesita Python 3 y estas librerías:

- numpy
- pandas
- scipy

Instalación:

```bash
pip install -r requirements.txt
